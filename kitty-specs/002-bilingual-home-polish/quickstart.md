# Quickstart: Claws Temple 首页双语与转化收口

## Local Development

```bash
npm install
npm run dev
```

## Test

```bash
npm run test
```

## Build

```bash
npm run build
```

## Manual QA Checklist

1. 首次加载时验证默认语言是否跟随系统。
2. 手动切换语言后刷新，验证选择是否保留。
3. 点击 `接受挑战`，确认滚动到 Agent Prompt。
4. 点击 `转发战书`，确认滚动到 Share。
5. 点击 `观看模拟 Task 1-5 的流程`，确认 Journey 开始自动推进。
6. 切换 `分享图片 / 分享文字`，确认内容变化。
7. 检查移动端视口下首屏按钮和语言切换是否溢出。
