# Contracts: Clawvard 风格首页重构与任务语义校正

本目录保存本轮 planning 阶段的内容 contract。

## Files

- `simulation-seed.schema.json`
  - 定义受控随机结果种子的结构
- `challenge-surface.schema.json`
  - 定义首页 / 分享区面向用户的结果展示载荷结构

## Usage

- `simulation-seed.schema.json` 用于约束 Hero、Share、Journey 的一致结果来源。
- `challenge-surface.schema.json` 用于约束首页成绩卡与分享战书的最终展示输入。
