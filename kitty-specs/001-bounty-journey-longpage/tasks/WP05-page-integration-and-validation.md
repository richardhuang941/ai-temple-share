### T021 - Compose `App` and `main` entry to mount all four sections

**Purpose**

Create the final single-page composition in the correct order and wire up the global stylesheet entry.

**Implementation Guidance**

1. Import the four section components and global styles through the final entry path.
2. Render the sections in the planned order:
   - Hero
   - Journey
   - Share
   - Agent Prompt
3. Keep composition simple and readable.
4. Do not insert extra route layers or multipage behavior.

**Validation**

- [ ] App renders as one continuous page
- [ ] Section order matches the specification

### T022 - Wire autoplay, anchors and share emphasis across the full page

**Purpose**

Connect page-level interactions so the longpage feels intentional rather than a stack of isolated blocks.

**Implementation Guidance**

1. Ensure Hero CTA targets the Journey area.
2. Connect the Journey sequence to the Share area emphasis if the design supports it.
3. Keep autoplay and reduced-motion behavior coherent at page level.
4. Avoid global state sprawl; prefer explicit prop flow into the composed page.

**Validation**

- [ ] Hero CTA reaches the intended section
- [ ] The share area is reachable and visually meaningful after the journey
- [ ] Reduced-motion behavior stays coherent across the page
